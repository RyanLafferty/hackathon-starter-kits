# express-js

## Technologies

* docker, docker-compose
* node, express

## Usage

### Build Application

```bash
docker-compose build
```

### Run Application

```bash
docker-compose up
```

### Test Application

```bash
docker-compose run express-js yarn test
```

### Generate Coverage For Application

```bash
docker-compose run express-js yarn test:coverage
```

### Access

```bash
localhost:8080
```
